# PRODIGY_WD_01

## Task 01 – Responsive Landing Page

This project is a responsive landing page created as part of **Task 01** of the **Prodigy InfoTech Web Development Internship**.

The main focus of this task is to design an **interactive navigation menu** that changes its style on scroll and hover, while remaining fixed and visible across the page.

---

## 🔹 Features

- Fixed navigation bar visible at all times
- Navigation menu changes background color when the page is scrolled
- Hover effects on navigation menu items
- Fully responsive design for mobile, tablet, and desktop screens
- Clean and modern UI using gradient shapes

---

## 🛠️ Technologies Used

- **HTML** – Structure of the webpage
- **CSS** – Styling, layout, responsiveness, and hover effects
- **JavaScript** – Scroll-based interactivity and dynamic style changes
